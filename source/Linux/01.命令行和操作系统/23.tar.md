# tar命令
```  
参数          作用 
-c          创建压缩文件 
-x          解开压缩文件 
-t          查看压缩包内有哪些文件 
-z          用Gzip压缩或解压 
-j          用bzip2压缩或解压 
-v          显示压缩或解压的过程 
-f          目标文件名 
-p          保留原始的权限与属性 
-P          使用绝对路径来压缩 
-C          指定解压到的目录

```
    
    用-v参数向用户不断显示压缩或解压的过程。-C参数用于指定要解压到哪个指定的目录

### 打包
    tar -czvf etc_bak.tar.gz /etc
    
    
### 解包
    tar -zxvf etc.tar.gz
