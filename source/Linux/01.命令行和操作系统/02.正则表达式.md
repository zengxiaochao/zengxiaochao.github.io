# 正则表达式

- 匹配一个电子邮箱
```
[a-z0-9_]+@[a-z0-9]+\.[a-z]+
```

- 匹配文本中的所有单词
```
( ?[a-zA-Z]+ ? )   #'?'匹配前后可能出现的空格，[a-zA-Z]代表一个或多个字母
```

- 匹配IP地址
```
[0-9]{1..3}\.[0-9]{1..3}\.[0-9]{1..3}\.[0-9]{1..3}
或
[[:digit:]]{1..3}\.[[:digit:]]{1..3}\.[[:digit:]]{1..3}\.[[:digit:]]{1..3}
```
